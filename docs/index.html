Functions

The package has three functions and a wrapper.

lm_euclid

lm_euclid fits a model with interactions and quadratic terms given a formula

For instance:

data <-
 fabricatr::fabricate(N = 40,
                      butter = rnorm(N), 
                      guns = rnorm(N),
                      rating  = -(butter^2 + (guns-.3)^2 + .2*rnorm(N)))

model <-  lm_euclid(rating ~ guns + butter, data)

The function simply creates the Euclidian formula and passes it to lm_robust.

You can provide any additional lm_robust functions you want as additional arguments, for instance, weights, cluster, blocks.

euclid_fits

euclid_fits is a simple helper to generate fitted values for a model given specified ranges and length

predictions_df <- 
  euclid_fits(formula = rating ~ guns + butter, model, data, lengths = c(2,3))

euclid_plot

euclid_plot call ggplot to plot the predicted fits and adds ideal points

predictions_df <- 
  euclid_fits(formula = rating ~ guns + butter, model, data, lengths = c(20,20))

euclid_plot(predictions_df, X = "guns", Y = "butter")

cjEuclid

cjEuclid wraps these functions and returns a list with the model, the fitted values, and the graph.

results <- cjEuclid(rating ~ guns + butter, data)
results$graph

Arguments

The functions include multiple arguments to allow graphing multiple dimensions. Here we illustrate plotting 2 dimensions with faceting by a third.

# Data has three dimensions and two observations per subject

data <-
 fabricatr::fabricate(
   ID = add_level(20),
   choice = add_level(2,
                      A = rnorm(N), B = rnorm(N), C = rnorm(N),
                      W = -(A^2 + (B-.3)^2 + (C --.67)^2) + .2*rnorm(N) + 3*A*B))


results <- 
  cjEuclid(W ~ A + B + C,
           fixed_effects = ~ ID,
           se_type = "stata", 
           data = data,
           X = "B",
           Y = "C",
           Row = "A",
           mins = c(-2, -2, -2),
           maxs = c(2, 2, 2),
           lengths = c(5, 25, 35),
           y_vals = c("little", "some", "much"),
           x_vals = c("This", "that", "there"))

results$graph

more ggplotting can be done later:

results$graph + 
  ylab("another label and a flip") + coord_flip() + 
  facet_grid(~ A, labeller = label_both)

Subgroup plots

Can be done by treating a subgroup as a variable, by stitching multiple plots together, or by running lm_euclid adn cj_Euclid on multiple groups and then applying the plot function with the group as a dimension.
